操作系统概念的学习-0-研究主题

发表于 2020-02-29 | 分类于 -操作系统理论

[操作系统概念的学习]0-研究主题

序言

研究主题

  • 概述

    ​ 概述章节主要的内容是: 解释操作系统的定义, 了解操作系统的功能以及设计与构造方式.

    ​ 主要讨论了操作系统的公共特性, 操作系统为用户提高的符合和操作系统为计算机系统操作员提供的功能.

    • 导论
    • 操作系统结构

  • 进程管理

    ​ 解释现代操作系统的核心概念—-进程与并发.

    ​ 进程是操作系统的工作单元, 一个系统由一组并发执行的进程组成, 包括: 系统进程, 用户进程.本章将会对一些进程处理进行讲解.

    • 进程
    • 多线程编程
    • 进程调度
    • 同步
    • 死锁

  • 内存管理

    ​ 本章主要讲解了操作系统的内存管理机制, 内存管理是将计算机内多个进程同时保存在内存中的管理方式, 它影响着CPU的利用率及其对用户的响应速度.

    • 内存管理策略
    • 虚拟内存管理

  • 存储管理

  • ​ 主要讲述了现代操作系统对文件系统,大容量存储器和I/O设备的处理方式.

    • 文件系统
    • 文件系统实现
    • 大容量存储器的结构
    • I/O系统

  • 保护与安全

    ​ 讨论操作系统的进程保护和安全处理机制.这是操作系统为了使不太的进程之间不会相互影响而采用的保护机制.

    • 系统保护
    • 系统安全

  • 案例研究

​ 研究了一些实际的操作系统的设计, 包括了Linux, Windows,FreeBSD和Mach

  • Linux系统
  • Windows 7
  • 有影响的系统
