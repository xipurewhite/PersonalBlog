章节内容

计算机体系结构的八大思想

面向摩尔定理的设计

概念

​ 计算机设计往往需要几年的时间, 而根据摩尔定理, 计算机的设计者们需要预测计算机在设计完成时的工艺水平而非设计开始时的.

摩尔定律

​ 单芯片集成电路的密集度每18个月到24个月会翻一倍以上.

抽象化设计

概念

​ 提高硬件和软件生产率的主要技术之一,通过抽象化来表达不同的设计层次, 在高层次中隐藏低层次的内容, 而是一个简单的模型.

并行处理

概念

​ 通过对程序进行并行化处, 将工作分配到多个不同的进程中, 是一种能够使计算机同时进行两个或多个处理的计算方法, 往往因为不同进程之间的联系, 并行处理无法自动化实现.

流水线处理

概念

​ 流水线处理是一种将程序多个指令重叠执行的准并行处理计算.
​ 在CPU中将N个不同功能的电路单元组成一条指令处理流水线, 然后将这条指令分为N步后由这些电路单元单独执行.和串行执行不同, 流水线处理将一条指令分给不同的电路单元顺序的完成不同步骤, 并且完成特定步骤的同时(当前指令并未处理完的时候), (由完成当前指令的子步骤的电路单元)对下一个指令进行处理.
​ 而在串行处理中, 一条指令的执行只有在上一条指令执行完成后才进行, 流水线处理是对指令执行过程中出现的空闲电路单元的利用.

加速大概率事件

概念

​ 对大概率事件进行优化远比对小概率事件的优化更能提高性能, 并且大概率事件往往比小概率事件更加简单, 从而也易于优化.

预测

概念

​ 在预测恢复执行代价不高并且预测的准确率相对较高的前提下, 通过猜测的方式提前开始某些任务, 会比知道这些任务确切会执行再启动才开始会快一些.

存储器层次

概念

​ 依据计算机不同功能的需求对存储器进行划分, 提出不同的存储器层次,从而解决存储器的速度和容量对性能和解题规模的影响矛盾

冗余利用

概念

​ 通过利用冗余部件来替代失效的部件并帮忙检测错误, 提高系统的可靠性.

计算机程序概念

软件层次

  • 顶层:应用软件
    • 数据库,网络浏览器,字处理工具等.
  • 中间层:系统软件
    • 操作系统,编译程序,加载程序和汇编程序等.
  • 底层:硬件

计算机硬件概念

计算机组成的经典组成部件

  • 输入设备

  • 为计算机提供信息的转置

  • 输出设备

  • 将计算机结果输出给用户的装置

  • 控制器
  • 运算器(数据通路)
  • 存储器

存储器

  • 主存储器(易失性存储器)
    • 动态随机访问存储器(DRAM)
    • 静态随机访问存储器(SRAM)
    • 缓存(cache memory)
      • 小而快,一般作为大而慢的存储器的缓冲器
      • 一般作为DRAM的缓冲器
  • 二级存储器(非易失性存储器)

  • 磁盘

  • 闪存

    • 写100 000~1 000 000次老化或损坏的弱点

硬件软件之间的接口

  • 指令集体系结构
    • 也称体系结构,低层次软件和引进之间的抽象接口
    • 包含编写正确运行的机器语言程序的全部信息:指令,寄存器,存储访问和I/O等
  • 应用二进制接口
    • 用户部分的指令和应用程序员调用的操作系统接口

性能

影响性能的因素

  • 响应时间(执行时间)

  • 计算机完成某项任务所需的总时间

  • 任务: 硬盘访问,内存读写,I/O活动,操作系统开销和CPU执行时间等

  • 吞吐率

  • 单位时间内完成的任务数量

性能的度量(CPU)

  • CPU执行时间

  • 执行某一任务在CPU上花费的时间

  • 用户CPU时间

  • 在程序本身花费的CPU时间

  • 系统CPU时间

  • 为执行程序而在操作系统上花费的时间

性能的描述

  • 系统性能

  • 空载系统的响应时间

  • CPU性能

  • 用户CPU时间

CPU性能

概念
  • 时钟周期
    • 计算机一个时间周期的时间,通常是处理器时钟,常数
  • 时钟频率
    • 时间周期的倒数
  • CPI
    • 执行每条指令所需的时间周期数的平均值
公式
  • CPU时钟周期数 = 程序的指令数 * CPI
  • CPU时间 = 程序的指令数 * CPI * 时钟周期数
  • CPU时间 = 程序的指令数 * CPI / 时钟频率
影响因素
  • 算法
    • 影响: 指令数, 可能的CPI
    • 算法决定源程序指令执行的数目
  • 编程语言
    • 影响: 指令数,CPI
    • 编程语言的语句必须翻译为指令,从而决定指令数
  • 编译程序
    • 影响: 指令数,CPI
    • 决定源程序到计算机指令的翻译过程
  • 指令集体系结构
    • 影响: 指令数,CPI时钟频率
    • 影响CPU性能所有3个方面
      • 完成某个功能所需的指令数
      • 每条指令的时钟周期数
      • 处理器的时钟频率
